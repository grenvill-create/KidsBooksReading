# KidsBooksReading (儿童英文原版绘本交互精读乐园) 🦒🛁🧼

欢迎来到 **KidsBooksReading** 项目！这是一个专门为 3-10 岁幼童打造的 **网页版原版英文绘本交互式精读应用**。

本项目以经典绘本 Usborne 的 《**Giraffe in the Bath**》（长颈鹿洗热水澡）为核心首发案例，深度融合了“**音频分割切片技术**”与“**渐进式交互伴读引导机制**”，并融入了极具童趣的“**极速自适应马卡龙 3D 泡泡设计系统**”。

---

## 🎨 特色核心：“极速自适应马卡龙 3D 泡泡设计系统”

本项目的前端界面均构建在这一套专门为小朋友定制的视觉系统上：
1. **治愈系马卡龙配色（Macaron Palette）**：
   - 摒弃生硬强烈的红绿蓝大原色，全面采用精心调配、饱和度柔和的糖果马卡龙色：天空蓝（Sky Blue）、泡泡粉（Playful Pink）、薄荷绿（Mint Green）、晨曦黄（Sunshine Yellow）、丁香紫（Lilac Purple）和蜜桔橙（Soft Orange），并配以温暖的奶油白色（Oatmeal Milk）背景，极力呵护孩子视力，营造温暖治愈的童趣绘本空间。
2. **微立体 squishy 3D 触觉卡片与气泡按钮（Tactile 3D Bubble System）**：
   - 所有按钮、单词气泡和信息卡片都被赋予了圆润的圆角（`border-radius: 20px~28px`）和加粗边框，并配备了偏移物理阴影。
   - 当鼠标**悬停（Hover）**时，卡片和按钮会顺滑上浮放大；当用户**点击/触控（Active/Click）**时，按钮会向下压缩（模拟物理气泡被按扁的质感），具有极其Q弹的解压交互手感。
3. **高弹性阻尼阻抗缓动曲线（Elastic Bounce Curve）**：
   - 全面抛弃传统线性或单纯渐进渐出动画，采用物理感极强的贝塞尔弹性曲线：`cubic-bezier(0.175, 0.885, 0.32, 1.275)`。在翻页、弹出单词卡片、礼花迸发时，均会产生微妙且极其精致的“Q弹回跳”微动画。
4. **极速自适应适配（Super-fast Responsive Adaptation）**：
   - 纯粹基于原生 Vanilla CSS 变量与 `flexbox/grid` 布局实现，不额外引入任何庞大的三方样式框架，从而保持了极简的 DOM 结构，使得页面能够以 **60fps** 的超凡帧率流畅运行在手机、iPad 平板或电脑上，多端完美自适应适配。

## ⚙️ 核心功能实现

### 1. 渐进式精读互动教学状态机（4-Stage Interactive Guided Reading）
针对绘本中的每一句话，应用在音频自动暂停时，带领小朋友流转过四个定制的学习阶段，构成一个完美的闭环精读学习：
- **【听】Listen 🎧**：高亮当前句子，引导孩子静心聆听原版配音，可点击重复播放。
- **【析】Analyze 💡**：自动暂停音频并向下滑入中文翻译，同时弹出“**语法句式小点拨**”，以极简可爱的语言为孩子剖析这一句里面的并列词、形容词或特色句式。
- **【词】Vocabulary 🎒**：句子中的所有“核心核心词汇”变为闪闪发亮的可点击单词。点击即可向上浮出精致的“单词泡泡卡”，包含：**精美 Emoji 辅助联想记忆**、**国际音标**、**简易中文释义**、**例句**，并且在点击单词时直接触发 **Web Speech API 纯正美音发音**！
- **【确】Confirm ✅**：展示一个不断跳动的彩色大泡泡按钮 `[ 我学会啦！下一句 ➔ ]`，孩子大声朗读一遍后，自主点击确认，即可流畅流转到下一句。

### 2. 智能 Web Audio 音频停顿对齐工具（ silence detection Auto-Aligner）
作为家长的“秘密武器”，我们在**家长控制面板**中集成了一个全网独创的 **Web Audio API 纯前端静音扫描断句打点工具**：
- **波形实时扫描**：无需任何后端服务器支持，家长上传或系统加载绘本 MP3 音频时，直接提取 PCM 通道数据。
- **智能静音停顿识别（Silence Detection）**：通过可调节的`静音音量阈值`和`最短停顿秒数`，算法能够毫秒级扫描音频中的空隙，自动算出每句话在音频中的 `audioStart` 和 `audioEnd` 时间戳。
- **可视化波形图**：在 Canvas 上渲染出声波动图与已截断句子的红线切片标记。
- **手动双滑块精调**：提供细致到 `0.1s` 的滑块和输入框，家长可以一边点击试听，一边手动校准每句话的起止秒数。
- **一键导出 JS**：微调完成后，点击一键复制，即可得到整本书符合 React 格式的 JSON 句子数据段，极速添加新绘本！

### 3. 读后趣味游戏大作战（Quiz Playground）
在绘本 22 句话全部精读打卡结束后，自动解锁游乐场环节：
- 包含三道专门根据绘本内容定制的多媒体选择题，考察阅读理解和细节捕捉能力。
- 答对时触发炫彩礼花雨（Confetti）并伴随星星音效，极大地建立小朋友的成就感。

### 4. 宝贝成长阅读报告 (Parent Mode Stats)
记录宝贝已获得的星星数、精读过的绘本字句数量，并在家长面板中附带一个“**复习生词本**”，以表格化卡片展示孩子曾点击攻克过的所有核心高频词。

## 🛠 开发与部署指南

### 本地运行
1. **安装依赖**：
   ```bash
   npm install
   ```
2. **启动本地开发服务器**：
   ```bash
   npm run dev
   ```
3. 打开浏览器访问 `http://localhost:5173/` 即可开始阅读！

### 打包构建
```bash
npm run build
```
这将在 `dist` 目录下输出高度压缩、极速加载的静态网页包。

### 部署到 GitHub Pages
项目已经在 `vite.config.js` 中配置了基准路径，支持直接一键打包部署。后续可通过公网网址在您的手机、iPad 上流畅交互！

```javascript
  sentences: [
    {
      id: "s1",
      text: "Giraffe is in her garden.",
      translation: "长颈鹿在她的花园里。",

      words: ["garden"]
    }
  ]
}
```
